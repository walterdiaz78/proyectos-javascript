Etiqueta pre

        La etiqueta pre en HTML significa "preformateado" (preformatted). Esta etiqueta se utiliza para mostrar texto con un formato específico, preservando tanto los espacios como los saltos de línea exactamente como se escriben en el código fuente HTML. Aquí tienes algunos puntos clave sobre la etiqueta 
:

            Formato preservado: El texto dentro de "pre" se muestra generalmente en una fuente de ancho fijo (monoespaciada) y preserva los espacios en blanco y saltos de línea.
            Uso común: Se utiliza frecuentemente para mostrar código, ASCII art, o cualquier texto que requiera un formato específico.
            Combinación con : A menudo se combina con la etiqueta "code" para mostrar bloques de código fuente.
            Estilo por defecto: Los navegadores suelen mostrar el contenido de "pre" en una fuente como Courier.
            Respeto del formato: A diferencia del texto normal en HTML, donde múltiples espacios se colapsan en uno solo, "pre" respeta todos los espacios.
